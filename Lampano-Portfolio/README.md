# Donna M. Lampano Portfolio

A cute-but-professional standalone HTML portfolio website for Donna M. Lampano.

## Theme

**Soft UI/UX, Research, and Documentation Portfolio**  
Focused on design, documentation, research, creative presentation, and user-centered academic projects.

## Main Difference from Aaron's Portfolio

This portfolio is not structured as a system/development portfolio. It uses:
- Different page flow
- Different section names
- Different project order
- Different color palette
- Different writing angle
- Different layout and component structure
- Stronger emphasis on UI/UX, research, visual presentation, and documentation

## Sections Included

- Hero
- Profile
- Strengths
- Selected Works
- College Journey
- Working Style
- Toolkit
- Contact

## How to Open

Open `index.html` directly in a browser.

## Where to Edit

- Main content: `index.html`
- Styling/theme: `src/styles.css`
- Page title/meta: `index.html`
- Profile image: `assets/images/profile/`
- Project images: `assets/images/projects/`
